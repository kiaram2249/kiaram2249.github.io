Garden-FP

By: Caron Voon, Robert Xu, and Kiara Matamoros

With my partners, we were able to make a game by using Kaboom and p5play. We used Kaboom to create the meun & the instrucations, while using p5play to make the mini-game.

While creating our game, we faced several challenges. One of the main issues was combining the menu with the mini-game. It didn't transition smoothly at first, which made the game feel disconnected. We also struggled with making certain object deadly without affecting others; for example, one object would work correctly, but when we added another, it wouldn't behave as expected. Another challenge was resizing images properly as some appeared to big or too small in the game. Despite these diffculties, we worked together to find solutions and improve our project.

This year in SEP taught me valuable skills beyond just coding. I learned how to search more effectively, debug problems, and keep trying different solutions. Working in a team helped me improve my communication, time management, and attention to detail, especially when meeting deadlines or fixing bugs.
